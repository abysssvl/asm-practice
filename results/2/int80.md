# Практична робота №2: Взаємодія з ОС через int 0x80

## 1. Що таке int 0x80
`int 0x80` — це інструкція програмного переривання в архітектурі x86, яка використовується для здійснення системних викликів (syscalls) у Linux. Вона перемикає процесор з режиму користувача в режим ядра, дозволяючи програмі запитувати послуги в ОС.

## 2. Основні системні виклики
Для виконання syscall через `int 0x80` використовуються наступні регістри:
- **EAX**: Номер системного виклику.
- **EBX, ECX, EDX**: Аргументи функції.

### Часто вживані функції:
1. **sys_write (EAX = 4)**: Вивід даних.
   - EBX: Файловий дескриптор (1 для stdout).
   - ECX: Вказівник на рядок.
   - EDX: Довжина рядка.
2. **sys_read (EAX = 3)**: Введення даних.
   - EBX: Файловий дескриптор (0 для stdin).
   - ECX: Буфер для запису даних.
   - EDX: Кількість байт.
3. **sys_exit (EAX = 1)**: Завершення програми.
   - EBX: Код виходу.
